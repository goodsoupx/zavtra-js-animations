# JavaScript Animations

A collection of beautiful and interactive JavaScript animations and effects.

## Features

- Smooth CSS animations
- Interactive JavaScript effects
- Responsive design
- Modern web standards

## Getting Started

### Installation

```bash
npm install
```

### Running Locally

```bash
npm start
```

Then open your browser to `http://localhost:8080`

## Examples

- Particle effects
- Smooth transitions
- Interactive animations
- Canvas-based effects
