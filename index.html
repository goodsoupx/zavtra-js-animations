This page tests the page preloader functionality. The preloader should appear when the page loads and disappear after the page is fully loaded (with a minimum display time of 1 second).

How to Test

To see the preloader again, you can:

  • Refresh the page (F5 or Ctrl+R)
  • Use hard refresh (Ctrl+Shift+R or Ctrl+F5)
  • Open DevTools and throttle network speed to see it longer
